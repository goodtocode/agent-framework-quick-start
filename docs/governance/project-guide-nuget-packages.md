# Project Guide: NuGet Packages

## Purpose
Define package selection and maintenance standards.

## Package Selection Criteria
- Prefer official Microsoft packages where suitable.
- Require active maintenance and community trust.
- Avoid overlapping dependencies with similar purpose.

## Review Process
- Perform open-source suitability review.
- Perform security and license review.
- Document rationale for non-standard package choices.

## Maintenance Process
- Review package updates regularly.
- Prioritize security updates.
- Validate upgrades with build/test checks before merge.
